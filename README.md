【项目说明书】大白Go：基于宇树Go2平台的全空间感知智能伴随系统

一、 项目简介：大白Go重塑人类行走的物理安全与心理尊严
从“工具辅助”向“器官延伸”的升维。
大白Go 是一款基于宇树Go2机器人平台深度开发的智能伴随系统。我们的研发灵感源于团队成员曾因意外导致短期单眼失明的真实经历，这段经历让我们深刻洞察到感知受限人群在城市出行中的三大核心痛点：
1. 感知断层（Safety Gap）： 传统盲杖与导盲犬的探测范围局限于地表，对于腰部以上的高位障碍（如低垂树枝、悬挂招牌）存在严重的感知盲区。
2. 交互疲劳（Interaction Fatigue）： 在嘈杂且信息过载的户外，依赖听觉的语音导航不仅干扰判断，更令用户感到心理焦虑。
3. 安全感缺失（Vulnerability）： 在陌生或开阔的公共空间，用户由于缺乏实时的环境掌控力，常处于高度紧绷的心理状态，极度缺乏“守护感”。
大白Go 通过“全高度感知”与“双向直觉交互”，填补了物理盲区，将“未知恐惧”转化为“笃定前行”。
值得关注的是，在实测阶段，大白Go 的价值边界展现出极强的向外兼容性： 它不仅是视障人士的导盲犬，更成为了都市人群追求“极致松弛感”的出行新标配。无论是追求前卫科技感的都市潮男，还是希望在城市探索中释放双手、保持优雅时尚的女性群体，或是对安全陪伴与即时响应有需求的银发群体人群，大白Go 都能提供一种跨越圈层的“伴随式”新生活体验。它让科技不再是冷冰冰的补偿工具，而是成为一种彰显生活态度、释放出行压力的潮酷智能单品。

二、 业务价值说明：
1. 安全平权：从“地面避障”到“全空间安全”的感知升级
对于视障群体或因伤导致感知受限的人群而言，安全不应是有门槛的选择。
现状局限： 传统盲杖与导盲犬的感知高度通常局限于腰部以下，导致视障者在面对低垂树枝、悬挂招牌等“高位危险”时仍处于弱势。
平权价值： 大白Go通过全高度感应模组，补全了这一生理盲区。它赋予了用户与普通人对等的、对 2 米全维度空间的感知权，让行走不再是一种风险博弈。
2. 交互升维：从“指令依赖”到“直觉反馈”的默契重塑
我们致力于消除技术带来的“社交标记感”，让交互回归自然。
体验优化： 相比于在公共场合大声播报的语音导航（这往往会强化用户的“特殊身份”），大白Go采用力反馈牵引杆。通过手柄微妙的推拉感和震动，用户能像牵着老友一样获得直觉引导。
社交融入： 特有的“安抚模式”让设备在咖啡馆、公交等室内场景能保持紧凑且静谧的姿态。这种设计有效降低了外界的侧目与用户的心理负担，协助用户更从容地融入主流社交环境。
3. 经济效益更具性价比：从“高昂维护”到“标准化服务”的效能革命
从经济学角度看，大白Go解决了传统导盲犬模式的结构性难题。
成本对比： 传统导盲犬训练成本通常高达 20 万人民币，且服役期仅 8-10 年，繁育与训练周期极长。
业务价值： 大白Go提供了高一致性、可快速规模化、且寿命周期内可持续升级的方案。它不仅降低了社会对导盲资源的投入门槛，更通过标准化的硬件服务，实现了比传统模式更高效、更经济的长期陪伴保障。
变现方式：
- To G (政府与社会公益)： 接入城市残联及适老化改造采购体系。通过**“以租代售”或“政府补贴”**模式，降低终端用户的首付压力，将其转化为公共基础设施的一部分，获取稳定的政府采购订单。
- To B (商业空间合作)： 与大型商场、机场、科技园区等公共场馆合作。由馆方投放作为**“共享陪伴设施”**，为视障或行动不便的访客提供标准化、无障碍的导览服务，提升商业空间的适老化评级与品牌形象。
- To C (个人消费市场)： 针对追求生活品质的家庭，提供**“硬件购买 + 算法订阅”**的会员制服务。除了基础的导盲功能，用户可订阅更高级的环境语义更新（如特定连锁超市的内部地图），实现长效的利润增长。

三、 AI 创新性说明：
我们将 AI 从单纯的“大脑”延伸到了“感官”与“肢体”，核心创新体现在以下三个层面：
1. 具身智能交互：基于力反馈的“灵觉牵引”算法
这是本项目最核心的交互创新。我们开发了一套自适应力反馈模型，将机械狗的路径规划轨迹转化为牵引杆上的物理矢量。
- AI 逻辑： 算法不只是让机器避障，而是实时计算最优路径与用户当前重心的偏差。
- 创新表现： AI 通过控制牵引杆产生微小的推力、拉力或特定频率的震动（如遇障碍回拉、引导转弯前推），让用户无需分心聆听语音，即可产生一种类似牵手行走的“生理直觉”，实现真实意义上的人机合一。
2. 环境语义理解：多模态 VLM 的深度场景应用
我们接入了先进的 VLM (视觉语言模型)，让大白 GoGo 从“看图”进化到“看懂生活”。
- AI 逻辑： 摄像头采集的画面不再仅仅是障碍物坐标，而是被转化为语义信息。
- 创新表现： 它能响应“带我去最近的咖啡馆”或“寻找附近的空座位”等模糊指令。AI 能够识别出复杂的社会化场景（如快递柜、电梯按键），并结合地图 API 规划出最具生活逻辑的行进路径，成为用户的“数字辅助视野”。
3. 智能安全中枢：主动防御与一键响应系统
我们将安全逻辑从“被动避障”升级为“主动防护”，并集成了紧急风险处理机制。
- AI 逻辑： AI 实时监测环境中的高位风险（数字胡须逻辑）及用户行进状态。
- 创新表现： * 高位风险预测： 针对树枝等悬挂物，AI 会提前通过手柄反馈进行阻拦式干预。
  - 异常一键直连： 结合手柄上的一键紧急呼救按钮，当 AI 监测到跌倒或用户触发求助时，系统将自动结合当前环境语义与地理坐标，联网呼叫紧急联系人或医疗机构，并在本地发起声光报警吸引路人关注，构建起最后一道安全防线。
4. 多模态情感交互：从“高效指令”到“情绪陪伴”
我们不仅对齐了智慧驾驶级的交互标准，更深度关怀用户的心理健康：
- 高效指令模式： 类似智能汽车，用户可通过“我要回家”、“去最近的地铁站”等简练指令，快速驱动 AI 决策，确保出行效率。
- 互怼/陪伴模式： 针对视障群体在社会交往中可能产生的压力，我们设计了具备性格色彩的对话逻辑。大白Go可以作为“互怼损友”或“关怀者”，通过语音与用户进行具有情感温度的互动（甚至包括解压的“互怼”），提供全天候的心理陪伴，缓解长期孤独感。
5. 未来生态协同：感知能力的跨端延伸
大白Go的底层架构具备极强的扩展性，未来将实现从“单一终端”向“穿戴设备集群”的演进：
- 多端联动： 我们计划接入 智能蓝牙眼镜 或 轻量化 AR/VR 设备。
- 协同效应： 眼镜负责采集用户第一视角（POV）的视觉信息，与机械狗的低位及全景雷达数据在云端融合。这种“头足联动”的架构，将为用户构建起一套覆盖更广、更具沉浸感的数字增强感官系统。

四、 技术实现说明：
核心逻辑：基于宇树原生强大性能，构建垂直领域的二次开发闭环。
- 硬件平台： 以宇树Go2作为移动底座，利用其卓越的运动稳定性与全地形通过能力。
- 传感矩阵： 在背部/肩膀集成高位传感模块（D435摄像头和3D激光雷达），补全高空感知盲区。
- 交互模组： 自研集成力反馈AI系统与紧急呼救按钮的“灵觉牵引杆”，牵引杆不仅仅是一根杆子，它内部集成了智能传动系统。当大白GoGo 发现危险需要停下或转弯时，它会通过杆子直接给主人的手掌一个轻微的“拉力”或“推力”
- 算法栈： 基于 ROS2 的运动控制 + 基于环境交互感知的多模态大模型 API处理环境语义。

五、 交互与设计说明：
1. 视觉设计：延续宇树原生“治愈系”工业美学
- 配色方案： 延续宇树原生的科技灰底色，延续宇树原生的暖灰色亲肤外壳。
- 形态语言： 整体线条圆润化处理，降低视觉攻击性，使其在公共场所（如商场、咖啡馆）显得更加和谐。
2. 语音交互：懂生活的“语义理解”
我们采用自然语音系统，支持模糊指令，让操作像聊天一样简单：
- 日常出行： “大白，带我去最近的瑞幸咖啡。”（大白启动，引导进入导航路径）
- 特定目标： “大白，帮我找个靠窗的空位。”（大白扫描环境，规划最优落座路线）
- 无声守护： 行进过程中，除必要的语音确认外，主要路况通过手柄传达，拒绝频繁的蜂鸣干扰，保持用户在公共空间的低调与体面。
- 状态指令： “我要回家。”（大白自动从安抚模式切换至跟随模式，并提示：“好的，我们出发吧。”）
3. 安全交互：双向联动的“一键呼救”
手柄顶端集成了物理级紧急呼救按钮，用于处理突发危机：
- 本地求援： 与此同时，机械狗会发出具有辨识度的警示音（非刺耳噪音，而是具备求助信号特征的声响），引起周围路人的注意并引导其提供帮助。
- - 信息同步： 按下按钮后，大白立即将实时GPS坐标、当前环境快照同步推送至紧急联系人及 120 平台。
4. 场景模式：融入环境的“安抚形态”
针对不同的社交深度，大白具备形态切换能力：
- 安抚模式 (Comfort Mode)： 在图书馆、办公室或咖啡馆等静谧空间，用户一键开启安抚模式。
- 低功耗守护： 机械狗会原地收缩折叠，降低高度，进入低功耗静音状态。它会紧贴主人脚边，不占用公共通道，以一种“隐形”的方式提供 24 小时随身守护。
